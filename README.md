# Library Management System (Backend)

A RESTful backend API for managing library books built with **Node.js, Express.js, and MongoDB**.
The system allows users to **add, view, update, delete, and search books** through REST API endpoints.

---

## 📦 Tech Stack

* Node.js
* Express.js
* MongoDB
* Mongoose
* Dotenv
* Render (Deployment)
* GitHub (Version Control)

---

## 📂 Project Features

* Add a new book
* Get all books
* Get book by ID
* Update book details
* Delete a book
* Search book by title
* Proper HTTP status codes
* Error handling middleware
* MongoDB schema validations

---

## 📘 Book Schema Fields

| Field            | Description                |
| ---------------- | -------------------------- |
| Book ID          | Auto-generated by MongoDB  |
| Title            | Book title (Required)      |
| Author           | Book author (Required)     |
| ISBN             | Unique book identifier     |
| Genre            | Book category              |
| Publisher        | Publisher name             |
| Publication Year | Year of publication        |
| Total Copies     | Total number of books      |
| Available Copies | Currently available copies |
| Shelf Location   | Location in library        |
| Book Type        | Reference / Circulating    |
| Status           | Available / Checked Out    |

---

## 🔗 API Endpoints

| Method | Endpoint                | Description          |
| ------ | ----------------------- | -------------------- |
| POST   | /books                  | Add a new book       |
| GET    | /books                  | Get all books        |
| GET    | /books/:id              | Get book by ID       |
| PUT    | /books/:id              | Update book details  |
| DELETE | /books/:id              | Delete book          |
| GET    | /books/search?title=xyz | Search book by title |

## ⚙️ Installation & Setup

Clone the repository

```
git clone https://github.com/Anii1801/Library_Management_044.git
```

Install dependencies

```
npm install
```

Create `.env` file

```
MONGO_URI=your_mongodb_connection_string
PORT=5000
```

Run the server

```
node index.js
```
